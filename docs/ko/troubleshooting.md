# 문제 해결

XBloom 공식 문제 해결 가이드를 이 통합의 에러 이벤트 타입에 매핑한
문서입니다. 머신이 BLE로 알람을 보고하면(cmd `0xFFFE` —
[protocol.md](protocol.md) 명령 테이블 참고) `event.error` 엔티티가 아래
타입 중 하나를 발화하며, 각 이벤트에는 원시 알람 `code`가 속성으로
포함되므로 자동화에서 대응하는 해결법과 함께 알림을 보낼 수 있습니다.

## 머신 알람 (이벤트 타입)

| 이벤트 타입 | 공식 명칭 | 해결법 |
| --- | --- | --- |
| `mismatched_power` | Mismatched power / 전원·전압 경고 | 머신 전압이 사용 국가 전압과 일치하는지 확인하고 올바른 전원 어댑터를 연결하세요. 전압이 같다면 다른 콘센트를 시도하세요. |
| `brewing_error` | Brewing error / 급수 경고¹ | 물탱크와 급수 라인에 물이 충분한지, 선택한 급수원이 켜져 있는지 확인하세요. 디스케일링을 실행하거나 급수구 막힘을 점검하세요. |
| `dock_moving_error` | Dock moving error | 도크 주변에 장애물이 없는지 확인하고 주변을 깨끗하게 유지하세요. 지속되면 전원을 뽑고 30초 후 다시 시도하세요. |
| `grinding_error` | Grinding error / 그라인더 과부하¹ | 그라인더에 이물질이나 낀 원두가 없는지 확인하세요. 오른쪽 노브를 빠르게 3번 눌러 재시작한 뒤 그라인더를 보정하고 다시 시도하세요. 과열 시 분쇄는 30초 이하, 세션 간 60초 이상 간격을 두세요. 매우 단단한 원두(Agtron 100+)는 에스프레소 분쇄에 권장되지 않습니다. |
| `scale_overload` | Scale overload | 2kg 한도를 초과하지 마세요. 계속 이상하면 저울 보정을 실행하세요(저울 페이지에서 중간 노브 3연타). |
| `upgrade_failed` | Upgrade failed | 펌웨어 업그레이드 중에는 블루투스를 켜고 머신 옆에 있으세요. 2~10분 소요됩니다. |

¹ "급수 경고", "그라인더 과부하", "넘침 감지/추출 대기"는 별도의 BLE
알람 id가 없습니다 — 머신 디스플레이는 구분하지만 BLE로는 위의 큰 분류
안에 포함되어 도착합니다 (2026-07-20 디컴파일 확인).

## 전용 신호가 있는 오류

| 이벤트 타입 | 공식 명칭 | 해결법 |
| --- | --- | --- |
| `water_shortage` | 물 부족 경고 (cmd 40522) | 급수원이 비었는지 확인하고 보충하세요. 보충도 보고되며, 해소 시 `water_shortage_cleared`가 발화합니다. |
| `no_beans` | 원두 부족 경고 (cmd 40517) | 이물질이나 낀 원두를 확인(그라인더를 가볍게 두드리기)한 뒤 원두를 추가하세요. |
| `abnormal_dose_or_water` | 넘침/도징 이상 (cmd 8204) | 시작 전 디스펜서 아래에 컵을 놓고, 추출 중 컵을 빼지 마세요. 푸어오버에 에스프레소 분쇄도는 권장되지 않습니다. |
| `abnormal_gear_position` | 그라인더 기어 위치 이상 (cmd 8203) | 그라인더 캘리브레이션을 실행하세요 (버튼 또는 `calibrate_xbloom_grinder`). |

## 그 외 공식 안내 (BLE 신호 없음)

- **블루투스 연결 문제** — 호스트의 블루투스가 켜져 있는지 확인하고,
  머신의 **오른쪽 노브를 빠르게 3번** 눌러 재시작하세요. 머신이 다시
  잡히면 통합이 자동으로 재연결합니다.
- **무부하 상태에서 무게가 크게 흔들림** — 공장 보호 테이프를 제거하고,
  멀티탭 대신 벽 콘센트에 직접 연결하세요(접지 안 된 기기가 물린 멀티탭은
  저울을 불안정하게 만듭니다). 저울 보정을 실행하세요(저울 페이지에서
  중간 노브 3연타).
